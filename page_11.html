Pros
of Using Rust on the Host Side

Memory Safety by Default

Prevents buffer overflows and use-after-free bugs at compile time through the ownership system

Safer Concurrency

Send/Sync discipline and compile-time checks ensure data-race freedom in multi-threaded code

Clear, Expressive APIs

Strong type system and modern error handling with Result/Option types create highly reliable interfaces

Performance Comparable to C/C++

Zero-cost abstractions plus LLVM optimizations (LTO) provide native performance without runtime overhead

Robust Ecosystem

Extensive crates for protocol parsing, serialization, and IPC backed by a strong community
